### 段落

+ HTML 段落是通过 ```<p>``` 标签进行定义的。
  
```(HTML)
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
```

注：浏览器会自动地在段落的前后添加空行。（</p> 是块级元素）

### 拆行

+ 不产生一个新段落的情况下进行换行（新行），用 ```<br>``` 标签

+ ```<br />``` 元素是一个空的 HTML 元素。由于关闭标签没有任何意义，因此它没有结束标签。

### 水平线

+ ```<hr />``` 标签在 HTML 页面中创建水平线。

+ hr 元素可用于分隔内容。

```(html)
<p>This is a paragraph</p>
<hr />
<p>This is a paragraph</p>
<hr />
<p>This is a paragraph</p>
```
