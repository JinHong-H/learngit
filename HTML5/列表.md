# 列表

### 常用标签

|标签|描述|
|--|--|
|\<ol>|有序列表|
|\<ul>|无序列表|
|\<li>|列表项|
|\<dl>|列表|
|\<dt>|自定义列表项目|
|\<dd>|自定义列表的描述|

实例：

```(html)
<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>菜鸟教程(runoob.com)</title>
</head>
<body>

<h4>无序列表:</h4>
<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>

<h4>有序列表：</h4>
<ol>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

<h4>自定义列表：</h4>
<dl>
<dt>Coffee</dt>
<dd>- black hot drink</dd>
<dt>Milk</dt>
<dd>- white cold drink</dd>
</dl>

</body>
</html>
```
